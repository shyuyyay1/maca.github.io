瑪卡是什麼植物？

它來自於祕魯山區，有南美洲人蔘之稱，其豐富的營養讓它成為時下熱門保健品之一，網路和實體店面熱銷，也由於其特殊的成分而被歸類為男性保健、增進雄風的新一代聖品。它是瑪卡。

瑪卡有什麼神奇的地方呢？

瑪卡的根中含有豐富的蛋白質和礦物質如鉀、鈣、鐵、鋅，可補充體力，根據聯合國農糧組織（FAO）的資料顯示，每100克瑪卡有15.4毫克的鐵、288毫克的鈣，當中有14％是蛋白質、78％是碳水化合物，是當地人的澱粉來源之一。

除了是糧食，其他可能功效包括減少焦慮、提升運動表現和肌肉能量，在2016年的一篇研究中也提到瑪卡可以改善學習力和記憶力，但實證研究數量不多。

而流傳瑪卡能壯陽，則是其中的成分瑪卡烯(manaenes)，具有提高男性性慾的效果。微笑藥局藥師廖偉呈指出，男性中年後可能因為工作忙碌或睪固酮下降而「性」趣缺缺，但瑪卡能在不影響血液賀爾蒙濃度狀況下，提升男性性欲，「如果改變賀爾蒙濃度，就有致癌的風險，」廖偉呈說。
